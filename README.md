# Angular 10 Movie-Recommendation application

## Prerequisite 
- Java
- Node
- MongoDB
- Intelij/eclips

## How to setup?
- Open database.txt file and run given two query is mongo shell or mongo GUI
- Start spring boot java server
- Start UI application with following commands
    - npm install
    - npm start
